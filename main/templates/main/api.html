Авторизация и получение токена

Делаем post-запрос на http://localhost:5000/auth/token/login/

В data прописываем существующий логин и пароль 

data = {
    'username': '<логин>',
    'password': '<пароль>',
}

Получается следующий код 

data = {
    'username': 'mylogin',
    'password': '123456',
}
url = 'http://localhost:5000/auth/token/login/'
token = requests.post(url, data=data)
